# iGEM 国际遗传基因工程设计大赛

## 竞赛简介

iGEM（International Genetically Engineered Machine）是全球最大的合成生物学竞赛，我在其中担任网页组组长。

## 工作内容

### 实验服务器平台

与网页组队员共同搭建校内比赛实验服务器平台：

- 实验数据管理系统
- 团队协作工具集成
- 实验进度可视化

### 队伍主页设计

完成网页美工与功能设计：

- **React + Next.js**：前端框架选型
- 响应式设计适配
- 交互动画与数据可视化
- 多语言支持

### 技术探索

在项目中还接触了 Haskell 函数式编程，拓宽了技术视野。

## 团队管理心得

作为网页组组长，最大的收获是：

- **高效沟通**：明确需求，减少误解
- **任务分配**：根据成员特长合理分工
- **协商与鼓励**：营造积极的团队氛围
- **进度管控**：定期 check-in，及时调整方案

## 核心收获

- 全栈 Web 开发能力
- 团队领导与项目管理
- 跨学科协作经验
