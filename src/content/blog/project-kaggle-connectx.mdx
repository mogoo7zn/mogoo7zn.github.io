## 技术方案

### DQN 方法

在探索初期，我首先尝试了 Deep Q-Network 方法：

- **状态编码**：将 6×7 的棋盘状态转换为张量表示
- **网络架构**：采用卷积神经网络提取特征，并通过全连接层输出动作价值
- **训练技巧**：引入经验回放与目标网络以提升训练稳定性

### AlphaZero 方法

为了追求更强的博弈能力，进一步实现了 AlphaZero 算法：

- **搜索算法**：使用 Monte Carlo Tree Search (MCTS) 进行动作规划
- **双头网络**：策略网络预测落子概率，价值网络评估当前局面胜率
- **自对弈**：通过自我博弈不断生成高质量的训练数据

### 排名表现

经过多轮迭代，Agent 取得了历史最高排名 **19/241 名**，这也验证了算法在复杂博弈环境下的有效性

## 关键技术细节

- **训练稳定性**：通过精细的学习率调度与梯度裁剪机制，避免模型发散
- **搜索效率**：优化 MCTS 的搜索逻辑，在有限时间内探索更多可能
- **博弈策略**：引入对手建模与开局库，提升 Agent 在实战中的应对能力

## 收获

- 深入掌握了 DQN、PPO、AlphaZero 等主流强化学习算法
- 初步理解了 Agent 开发的核心思路，即如何从零开始构建一个智能体
