# pUC19-GAL4质粒线性化、URA3标签获取，pUC19-GAL4-URA3质粒构建#2025-07-19
实验人：陈梓凡、张景博
实验目的：构建pUC19-GAL4-URA3质粒
实验步骤：
+ 一、线性化pUC19-GAL4质粒、URA3标签获取
	+ pUC19-GAL4以pUC19-Gal4-linear F/R，p426-SNR52p-gRNA.CAN1.Y-SUP4t以Ura3-pUC19-Gal4 F/R为引物，进行PCR反应，获得线性化的载体和对应的标签。反应体系见TaKaRa PrimeSTAR Max(R045)试剂盒
+ 二、电泳，胶回收
	+ 胶回收见诺唯赞FastPure Gel DNA Extraction Mini Kit试剂盒
	+ 产物浓度：
		+ 线性化pUC19-GAL4：233.5 ng/uL
		+ URA3标签：73.0 ng/uL
+ 三、重组
	+ 将线性化pUC19-GAL4与Ura3-pUC19-Gal4（见07-13pUC19-GAL4-URA3、pUC19-GAL80-HIS3、pUC19-STE2-URA3质粒的构建）进行单片段同源重组。共构建2个重组体系。重组体系见ClonExpress II One Step Cloning Kit试剂盒
+ 四、PCR检验
	+ 以Gal4-pUC19 F/R为引物，重组产物作为模板进行PCR反应。反应体系见诺唯赞2XRapid Taq Master Mix Kit试剂盒
	+ 经过电泳，1号重组体系疑似阳性
