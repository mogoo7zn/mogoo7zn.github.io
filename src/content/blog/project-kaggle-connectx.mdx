# Kaggle ConnectX 强化学习竞赛

## 竞赛简介

Kaggle ConnectX 是一个重力四子棋（Connect Four）的强化学习竞赛，参赛者需要开发智能 Agent 与其他选手对战。

## 技术方案

### DQN 方法

首先尝试了 Deep Q-Network 方法：

- 状态编码：6×7 棋盘状态的张量表示
- 网络架构：卷积神经网络 + 全连接层
- 经验回放与目标网络

### AlphaZero 方法

进一步实现了 AlphaZero 算法：

- Monte Carlo Tree Search (MCTS)
- 策略网络 + 价值网络
- 自对弈训练流程

### 排名表现

历史最高排名 **19/241 名**，验证了算法的有效性。

## 关键技术细节

- 训练稳定性：学习率调度与梯度裁剪
- MCTS 搜索效率优化
- 对手建模与开局库

## 核心收获

- 掌握基本强化学习算法（DQN、PPO、AlphaZero）
- 理解 Agent 开发的核心思路
- Kaggle 竞赛实战经验
